# MONO 진단검사 동적 컨텐츠 생성용 코멘트 데이터베이스

## 📋 문서 개요

본 문서는 MONO 진단검사 결과지의 다음 섹션을 동적으로 생성하기 위한 코멘트 데이터베이스입니다:
- **06. 영역별 상세 분석** - 4개 영역별 강점/약점 코멘트
- **10. 3개월 맞춤 학습 로드맵** - 취약 영역 기반 학습 계획

**대상 학생:** 중1-1 과정 이수자  
**출제 범위:** 소인수분해, 정수와 유리수, 문자와 식, 좌표평면과 그래프, 정비례와 반비례  
**난이도 특징:** 중등 기본 + 사고력 중심 (고난도 27%)

## 🔢 영역 1: 수와 연산 (1~8번, 31.5점)

### Excellent (T-Score 70+)

**코멘트:**
```
소인수분해와 정수의 성질을 완벽하게 이해하고 있습니다. 약수의 개수 구하기, 최대공약수와 최소공배수 활용 능력이 뛰어나며, 복잡한 분수 계산도 정확합니다. 특히 약수와 배수의 관계를 이용한 사고력 문제도 잘 해결하고 있어, 중2 과정으로 자연스럽게 확장할 수 있는 탄탄한 기초를 갖추었습니다.
```

**학습 조언:**
```
현재 수준을 유지하며 유리수와 순환소수로 확장하세요. 정수론 심화 문제(완전제곱수, 약수의 합 등)나 수학 경시대회 기출문제에 도전해보세요.
```

### Good (T-Score 60~69)

**코멘트:**
```
소인수분해의 기본 개념은 확실히 이해하고 있으며, 최대공약수와 최소공배수를 구하는 능력도 우수합니다. 다만 약수의 개수를 구하거나 여러 조건이 복합된 문제에서 일부 실수가 발생합니다. 정수와 유리수의 사칙연산은 대체로 정확합니다.
```

**학습 조언:**
```
소인수분해를 활용한 약수 개수 구하는 공식을 확실히 익히세요. 복잡한 문제는 단계를 나누어 차근차근 접근하는 연습이 필요합니다.
```

### Average (T-Score 40~59)

**코멘트:**
```
기본적인 소인수분해는 가능하나, 이를 활용하는 문제에서 어려움을 겪고 있습니다. 정수의 성질(홀수, 짝수, 약수 등)에 대한 이해가 부족하며, 분수 계산에서도 실수가 많습니다.
```

**학습 조언:**
```
소인수분해의 의미를 다시 정리하고, 약수와 배수의 관계를 확실히 이해하세요. 교과서의 기본 문제부터 충분히 연습한 후 응용 문제로 넘어가세요.
```

### Weak (T-Score 30~39)

**코멘트:**
```
소인수분해의 기본 개념이 불완전하며, 정수의 사칙연산에서도 오류가 많습니다. 약수와 배수의 개념이 혼동되고 있으며, 최대공약수와 최소공배수 구하기가 어렵습니다.
```

**학습 조언:**
```
소수와 합성수의 개념부터 다시 학습하세요. 소인수분해를 정확히 하는 연습을 반복하고, 기본 문제를 충분히 풀어본 후 다음 단계로 넘어가세요.
```

### Critical (T-Score 30 미만)

**코멘트:**
```
수와 연산의 기초 개념이 매우 부족합니다. 소수의 정의조차 불명확하며, 기본적인 정수 계산도 어려워하고 있습니다. 초등 과정의 약수와 배수 개념 재정립이 필요합니다.
```

**학습 조언:**
```
초등 6학년 과정부터 재학습이 필요합니다. 약수와 배수의 기본 개념을 천천히 다시 배우고, 1:1 개별 지도를 통해 기초부터 차근차근 다져야 합니다.
```

### 틀린 문항 패턴별 추가 코멘트

**약수 개수 문제 오답 (1번, 2번 유형):**
```
완전제곱수나 인수의 조합으로 약수 개수를 구하는 문제에서 오답이 나왔습니다. 소인수분해 결과에서 약수 개수를 구하는 공식 (a+1)(b+1)(c+1)...을 정확히 익히세요.
```

**복잡한 분수식 오답 (4번, 7번 유형):**
```
연분수나 복잡한 분수식 계산에서 실수했습니다. 계산 순서를 정확히 지키고, 각 단계를 꼼꼼히 확인하며 풀이하세요.
```

**최대공약수/최소공배수 활용 오답 (5번, 6번, 8번 유형):**
```
최대공약수나 최소공배수를 활용한 응용 문제에서 어려움을 겪었습니다. 문제 조건을 정확히 파악하고, 어떤 개념을 사용해야 하는지 판단하는 연습이 필요합니다.
```

## ➕ 영역 2: 식의 계산 (9~12번, 15.5점)

### Excellent (T-Score 70+)

**코멘트:**
```
문자와 식의 개념을 완벽히 이해하고 있으며, 일차식의 덧셈, 뺄셈, 대입 계산이 정확합니다. 복잡한 식의 값을 구하는 문제도 단계적으로 잘 해결하고 있습니다. 특히 패턴을 인식하고 일반화하는 능력이 뛰어나 중2의 다항식 계산으로 자연스럽게 확장할 수 있습니다.
```

**학습 조언:**
```
현재 수준을 유지하며 중2의 단항식과 다항식 계산으로 확장하세요. 등차수열의 합이나 패턴 인식 문제 등 고급 사고력 문제에 도전해보세요.
```

### Good (T-Score 60~69)

**코멘트:**
```
일차식의 기본 계산은 정확하게 수행합니다. 동류항 정리와 간단한 대입 계산은 잘하고 있으나, 복잡한 식의 값을 구하거나 패턴을 인식하는 문제에서 일부 어려움이 있습니다.
```

**학습 조언:**
```
복잡한 식도 단계별로 차근차근 정리하는 연습이 필요합니다. 특히 음수와 분수가 포함된 대입 문제를 반복 학습하세요.
```

### Average (T-Score 40~59)

**코멘트:**
```
문자를 사용한 식의 기본 개념은 이해하고 있으나, 동류항 정리나 식의 값 구하기에서 실수가 많습니다. 특히 부호 처리가 미흡하며, 복잡한 대입 문제에서 어려움을 겪고 있습니다.
```

**학습 조언:**
```
동류항의 개념을 확실히 이해하고, 일차식의 덧셈과 뺄셈을 반복 연습하세요. 대입 시 부호를 정확히 처리하는 연습이 필요합니다.
```

### Weak (T-Score 30~39)

**코멘트:**
```
문자와 식의 기본 개념이 불완전합니다. 문자가 나타내는 의미를 정확히 파악하지 못하며, 일차식의 기본 계산에서도 많은 오류가 발생합니다.
```

**학습 조언:**
```
문자의 의미와 대입의 개념부터 다시 학습하세요. 간단한 식의 계산부터 충분히 연습하고, 천천히 단계를 높여가세요.
```

### Critical (T-Score 30 미만)

**코멘트:**
```
문자와 식의 기초 개념이 거의 없습니다. 문자를 포함한 식을 수로 보지 못하며, 기본적인 대입조차 어려워하고 있습니다.
```

**학습 조언:**
```
문자와 식의 도입부터 천천히 재학습하세요. 구체적인 수로 시작하여 점차 문자로 일반화하는 과정을 단계적으로 학습해야 합니다.
```

### 틀린 문항 패턴별 추가 코멘트

**음수/분수 대입 오답 (9번, 10번 유형):**
```
음수나 분수를 대입하는 과정에서 실수했습니다. 대입 시 괄호를 사용하고, 부호 규칙을 정확히 적용하세요.
```

**패턴 인식 오답 (11번, 12번 유형):**
```
수열이나 패턴을 인식하는 문제에서 어려움을 겪었습니다. 규칙을 찾아 식으로 나타내는 연습이 필요합니다. 이는 중3~고1 수준의 고난도 문제이므로, 기본을 다진 후 천천히 접근하세요.
```

## 📐 영역 3: 방정식 (13~20번, 32.5점)

### Excellent (T-Score 70+)

**코멘트:**
```
일차방정식의 개념을 완벽히 이해하고 있으며, 다양한 유형의 방정식을 정확하게 해결합니다. 항등식, 문장제 방정식, 절댓값 방정식, 심지어 가우스 기호 방정식까지 폭넓게 대응하고 있습니다. 좌표평면에서의 도형 문제도 잘 해결하여 중2의 연립방정식으로 자연스럽게 확장할 수 있습니다.
```

**학습 조언:**
```
현재 수준을 유지하며 중2의 연립방정식과 일차부등식으로 확장하세요. 특수한 방정식(무리방정식, 고차방정식 등) 문제에도 도전해보세요.
```

### Good (T-Score 60~69)

**코멘트:**
```
기본적인 일차방정식 풀이는 매우 정확합니다. 다항식 정리나 간단한 문장제도 잘 해결하고 있으나, 절댓값이나 가우스 기호가 포함된 특수 방정식, 좌표평면 도형 문제에서 일부 어려움이 있습니다.
```

**학습 조언:**
```
특수한 유형의 방정식(절댓값, 가우스 기호)은 경우를 나누어 푸는 연습이 필요합니다. 좌표평면에서의 도형 문제도 추가 학습하세요.
```

### Average (T-Score 40~59)

**코멘트:**
```
간단한 일차방정식은 풀 수 있으나, 복잡한 계수나 분수가 포함된 방정식에서 실수가 많습니다. 항등식의 개념이 불완전하며, 문장제 방정식을 식으로 세우는 데 어려움을 겪고 있습니다.
```

**학습 조언:**
```
방정식의 기본 풀이 과정을 정확히 익히세요. 분모를 없애는 방법, 괄호를 푸는 방법 등을 반복 연습하고, 항등식의 개념을 다시 정리하세요.
```

### Weak (T-Score 30~39)

**코멘트:**
```
일차방정식의 기본 개념이 불완전합니다. 등식의 성질을 제대로 이해하지 못하고 있으며, 이항이나 양변을 같은 수로 나누는 과정에서 많은 오류가 발생합니다.
```

**학습 조언:**
```
등식의 성질부터 다시 학습하세요. 가장 간단한 형태의 방정식(ax=b)부터 충분히 연습한 후 단계적으로 복잡한 방정식으로 넘어가세요.
```

### Critical (T-Score 30 미만)

**코멘트:**
```
방정식의 기초 개념이 거의 없습니다. 등식과 방정식의 차이조차 이해하지 못하고 있으며, 미지수의 개념도 불명확합니다.
```

**학습 조언:**
```
등식의 개념부터 천천히 재학습하세요. 구체적인 수를 사용한 등식부터 시작하여, 점차 미지수를 포함한 방정식으로 확장해야 합니다. 1:1 지도가 필요합니다.
```

### 틀린 문항 패턴별 추가 코멘트

**항등식 오답 (15번 유형):**
```
항등식 문제에서 오답이 나왔습니다. 항등식은 x의 값에 관계없이 항상 성립하는 식이라는 개념을 이해하고, 계수 비교 방법을 익히세요.
```

**절댓값/가우스 기호 방정식 오답 (17번, 18번 유형):**
```
절댓값이나 가우스 기호가 포함된 방정식에서 어려움을 겪었습니다. 이는 중3~고1 수준의 고난도 문제로, 경우를 나누어 푸는 방법을 학습해야 합니다.
```

**좌표평면 도형 오답 (20번 유형):**
```
좌표평면에서 도형의 넓이를 구하는 문제에서 실수했습니다. 좌표를 이용한 넓이 공식을 익히고, 기본 도형 문제를 반복 연습하세요.
```


### Good (T-Score 60~69)

**코멘트:**
```
정비례와 반비례의 기본 개념은 잘 이해하고 있으며, 간단한 그래프 해석도 정확합니다. 함수식을 구하거나 그래프를 그리는 능력도 우수하나, 도형과 결합된 복잡한 문제에서 일부 어려움이 있습니다.
```

**학습 조언:**


### Average (T-Score 40~59)

**코멘트:**
```
정비례와 반비례의 정의는 알고 있으나, 그래프 해석이나 함수식 구하기에서 어려움을 겪고 있습니다. 비례상수의 의미를 정확히 파악하지 못하고 있으며, 그래프에서 정보를 읽어내는 능력이 부족합니다.
```

**학습 조언:**
```
정비례(y=ax)와 반비례(y=a/x)의 기본 그래프 특징을 확실히 익히세요. 비례상수가 그래프에서 어떤 의미인지 이해하고, 그래프 그리기를 반복 연습하세요.
```

### Weak (T-Score 30~39)

**코멘트:**
```
함수의 기본 개념이 불완전합니다. 정비례와 반비례의 차이를 명확히 구분하지 못하고 있으며, 함수식과 그래프의 관계 이해가 부족합니다.
```

**학습 조언:**
```
함수의 정의부터 다시 학습하세요. 정비례는 원점을 지나는 직선, 반비례는 쌍곡선이라는 기본 특징을 확실히 이해하고, 간단한 문제부터 연습하세요.
```

### Critical (T-Score 30 미만)

**코멘트:**
```
함수의 기초 개념이 거의 없습니다. 좌표평면에서 점을 찍는 것조차 어려워하며, 함수의 의미를 전혀 이해하지 못하고 있습니다.
```

**학습 조언:**
```
좌표평면의 개념부터 재학습하세요. x축, y축의 의미를 이해하고, 점의 좌표를 읽고 찍는 연습부터 시작해야 합니다.
```

### 틀린 문항 패턴별 추가 코멘트

**그래프 해석 오답 (21번, 22번 유형):**
```
두 함수의 그래프를 해석하는 문제에서 오답이 나왔습니다. 그래프에서 조건을 만족하는 구간이나 값을 찾는 연습이 필요합니다.
```

**도형과 함수 결합 오답 (23번, 24번, 25번 유형):**
```
함수와 도형이 결합된 고난도 문제에서 어려움을 겪었습니다. 좌표평면에서 넓이를 구하는 공식을 익히고, 함수의 교점을 활용하는 문제를 집중 학습하세요.
```

## 🎯 득점 향상 예측 공식 (MONO)

### 기본 공식

```
예상 득점 = (영역 배점) × (목표 달성률) × (학습 기간 계수)

학습 기간 계수:
- 1~2개월: 0.5~0.6
- 2~3개월: 0.7~0.8
- 3개월+: 0.9~1.0

목표 달성률:
- Critical → Weak: 30~40%
- Weak → Average: 50~60%
- Average → Good: 70~80%
- Good → Excellent: 85~95%
```

### 예시 계산 (MONO)

**학생 상황:** 방정식 영역 T-Score 35 (Weak), 배점 32.5점, 현재 정답률 40%

**1단계 목표:** Weak → Average (정답률 55% 달성)

**예상 득점 향상:**
```
현재 득점: 32.5 × 0.40 = 13.0점
목표 득점: 32.5 × 0.55 = 17.9점
득점 향상: 17.9 - 13.0 = 4.9점 (보수적)

또는

현재 득점: 32.5 × 0.40 = 13.0점
목표 득점: 32.5 × 0.60 = 19.5점
득점 향상: 19.5 - 13.0 = 6.5점 (낙관적)

→ 표시: +5~7점
```

## 🔧 구현 시 고려사항 (MONO)

1. **우선순위 로직**
   ```python
   # MONO는 배점이 비슷한 영역이 많아 T-Score 우선
   def get_priority_areas_mono(results):
       areas = []
       for area, data in results.items():
           if data['t_score'] < 40:
               priority = 1
           elif data['t_score'] < 60:
               priority = 2
           else:
               priority = 3
           
           areas.append({
               'name': area,
               'priority': priority,
               'weight': data['total_score'],
               't_score': data['t_score']
           })
       
       # 우선순위 -> 배점 순으로 정렬
       return sorted(areas, key=lambda x: (x['priority'], -x['weight']))
   ```

   ```python
   def analyze_mono_patterns(area, wrong_items):
       patterns = []
       
       # 약수 개수 오답
       if area == "수와 연산" and (1 in wrong_items or 2 in wrong_items):
           patterns.append("약수개수_오답")
       
       # 항등식 오답
       if area == "방정식" and 15 in wrong_items:
           patterns.append("항등식_오답")
       
       # 가우스 기호/절댓값 오답  
       if area == "방정식" and (17 in wrong_items or 18 in wrong_items):
           patterns.append("특수방정식_오답")
       
       # 패턴 문제 오답
       if area == "식의 계산" and (11 in wrong_items or 12 in wrong_items):
           patterns.append("패턴인식_오답")
       
       # 도형과 함수 결합 오답
       if area == "함수" and set(wrong_items) & {23, 24, 25}:
           patterns.append("도형함수결합_오답")
       
       return patterns
   ```

4. **득점 향상 예측 (MONO 특화)**
   ```python
   def predict_mono_score_gain(area, current_score, target_level, period):
       area_weights = {
           "방정식": 32.5,
           "수와 연산": 31.5,
           "함수": 20.5,
           "식의 계산": 15.5
       }
       
       area_weight = area_weights[area]
       current_rate = current_score / area_weight
       
       # 목표 달성률
       target_rates = {
           'critical_to_weak': 0.35,
           'weak_to_average': 0.55,
           'average_to_good': 0.75,
           'good_to_excellent': 0.90
       }
       
       period_coefs = {
           '1-2개월': 0.55,
           '2-3개월': 0.75,
           '3개월+': 0.95
       }
       
       target_rate = target_rates[target_level]
       period_coef = period_coefs[period]
       
       estimated_gain = area_weight * (target_rate - current_rate) * period_coef
       
       # 보수적/낙관적 범위
       conservative = estimated_gain * 0.8
       optimistic = estimated_gain * 1.2
       
       return f"+{int(conservative)}~{int(optimistic)}점"
   ```
