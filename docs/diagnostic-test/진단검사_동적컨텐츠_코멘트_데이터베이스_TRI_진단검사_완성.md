# TRI 진단검사 동적 컨텐츠 생성용 코멘트 데이터베이스

## 📋 문서 개요

본 문서는 TRI 진단검사 결과지의 다음 섹션을 동적으로 생성하기 위한 코멘트 데이터베이스입니다:
- **06. 영역별 상세 분석** - 4개 영역별 강점/약점 코멘트
- **10. 3개월 맞춤 학습 로드맵** - 취약 영역 기반 학습 계획

**대상 학생:** 중3-1 과정 이수자  
**출제 범위:** 중3-1 + 공통수학1 이차함수  
**난이도 특징:** 고등 수학 준비도 측정 (고난도 46%, 고1 내용 포함)

## 🔢 영역 1: 실수와 연산 (1~8번, 31.0점)

### Excellent (T-Score 70+)

**코멘트:**
```
제곱근과 무리수의 개념을 완벽하게 이해하고 있습니다. 복잡한 근호 계산과 분모의 유리화를 정확하게 수행하며, 근과 계수의 관계를 활용한 고난도 문제도 잘 해결합니다. 특히 지수법칙과 제곱근을 결합한 사고력 문제(6, 7번)도 능숙하게 풀어내어 고등 수학의 복소수와 지수함수로 자연스럽게 확장할 수 있는 탄탄한 기초를 갖추었습니다.
```

**학습 조언:**
```
현재 수준을 유지하며 고1의 복소수, 거듭제곱근으로 확장하세요. 특히 고등 수학 심화 문제(지수방정식, 로그방정식)에 도전하고, 수학 경시대회 기출문제를 풀어보세요.
```

### Good (T-Score 60~69)

**코멘트:**
```
기본적인 제곱근 계산과 분모의 유리화는 정확합니다. 무리수의 크기 비교와 사칙연산을 잘 수행하고 있으나, 근과 계수의 관계나 대칭식을 활용하는 고난도 문제(7, 8번)에서 일부 어려움을 겪습니다.
```

**학습 조언:**
```
근과 계수의 관계를 완벽히 익히고, 대칭식 변형 연습을 하세요. 특히 a+b+c, ab+bc+ca, abc 조건에서 다른 식을 구하는 고1 수준 문제를 학습하세요.
```

### Average (T-Score 40~59)

**코멘트:**
```
기본적인 제곱근은 이해하고 있으나, 복잡한 근호 계산이나 분모의 유리화에서 실수가 많습니다. 가우스 기호 활용 문제나 지수법칙과 결합된 문제에서 어려움을 겪고 있습니다.
```

**학습 조언:**
```
제곱근의 기본 성질(√a×√b = √(ab), √a/√b = √(a/b))을 확실히 익히세요. 분모의 유리화 과정을 단계별로 연습하고, 중3 교과서 기본 문제부터 충분히 풀어보세요.
```

### Weak (T-Score 30~39)

**코멘트:**
```
제곱근의 기본 개념이 불완전합니다. 근호 계산에서 많은 오류가 발생하며, 무리수와 유리수의 구분도 명확하지 않습니다. 분모의 유리화 방법을 잘 모르고 있습니다.
```

**학습 조언:**
```
제곱근의 정의부터 다시 학습하세요. √4 = 2, √9 = 3 같은 기본부터 시작하여 √2, √3 등 무리수 개념을 확실히 이해하세요. 중3 교과서를 처음부터 정독하세요.
```

### Critical (T-Score 30 미만)

**코멘트:**
```
실수의 기초 개념이 매우 부족합니다. 제곱근의 의미를 이해하지 못하고 있으며, 근호가 포함된 계산을 거의 하지 못합니다. 중2 과정의 유리수 계산도 재점검이 필요합니다.
```

**학습 조언:**
```
중2 유리수 과정부터 재학습이 필요합니다. 제곱의 개념을 확실히 이해한 후 제곱근으로 넘어가세요. 1:1 개별 지도를 통해 기초부터 차근차근 다져야 합니다.
```

**학습 조언:**

```

**학습 조언:**

```
간단한 이차방정식은 풀 수 있으나, 근과 계수의 관계를 활용하는 문제나 특수 조건이 주어진 방정식에서 어려움을 겪습니다. 판별식의 의미도 불완전합니다.

**학습 조언:**
```

**코멘트:**

**학습 조언:**

**코멘트:**

```

**학습 조언:**
```
일차방정식부터 재학습하세요. 방정식의 해가 무엇을 의미하는지 이해한 후, 간단한 이차방정식(x²=4, x²-5=0 등)부터 천천히 접근하세요.
```

## 🎯 득점 향상 예측 공식 (TRI)

### 기본 공식

```
예상 득점 = (영역 배점) × (목표 달성률) × (학습 기간 계수)

학습 기간 계수:
- 1~2개월: 0.45~0.55 (TRI는 난이도가 높아 계수 낮음)
- 2~3개월: 0.65~0.75
- 3개월+: 0.85~0.95

목표 달성률:
- Critical → Weak: 30~40%
- Weak → Average: 50~60%
- Average → Good: 70~80%
- Good → Excellent: 85~95%
```

### 예시 계산 (TRI)

**학생 상황:** 이차함수 영역 T-Score 38 (Weak), 배점 35.5점, 현재 정답률 40%

**1단계 목표:** Weak → Average (정답률 55% 달성)

**예상 득점 향상:**
```
현재 득점: 35.5 × 0.40 = 14.2점
목표 득점: 35.5 × 0.55 = 19.5점
득점 향상: 19.5 - 14.2 = 5.3점 (보수적)

또는

현재 득점: 35.5 × 0.40 = 14.2점
목표 득점: 35.5 × 0.60 = 21.3점
득점 향상: 21.3 - 14.2 = 7.1점 (낙관적)

→ 표시: +5~7점
```

## 🔧 구현 시 고려사항 (TRI)

1. **우선순위 로직**
   ```python
   # TRI는 이차함수 배점이 압도적으로 높음
   def get_priority_areas_tri(results):
       areas = []
       for area, data in results.items():
           # 이차함수는 특별 가중치 부여
           if area == "이차함수":
               weight_multiplier = 1.5
           else:
               weight_multiplier = 1.0
           
           if data['t_score'] < 40:
               priority = 1
           elif data['t_score'] < 60:
               priority = 2
           else:
               priority = 3
           
           areas.append({
               'name': area,
               'priority': priority,
               'weight': data['total_score'] * weight_multiplier,
               't_score': data['t_score']
           })
       
       return sorted(areas, key=lambda x: (x['priority'], -x['weight']))
   ```

       patterns = []
       
       # 근과 계수 고급 활용 오답
       if area == "실수와 연산" and (7 in wrong_items or 8 in wrong_items):

       if area == "다항식" and 9 in wrong_items:
           patterns.append("고등인수분해공식")
