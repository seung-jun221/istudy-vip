## 🔢 영역 1: 수와 연산 (Numbers and Operations)

### Excellent (T-Score 70+)

**코멘트:**
```
유리수와 순환소수에 대한 개념이 완벽하며, 복잡한 분수 계산도 정확하게 수행합니다. 소인수분해와 최대공약수/최소공배수 활용 능력이 뛰어나 고등 과정의 실수 체계로 자연스럽게 확장할 수 있는 탄탄한 기초를 갖추고 있습니다.
```

**학습 조언:**
```
현재 수준을 유지하며 고등 수학의 실수 체계(무리수, 절댓값, 제곱근)로 확장 학습을 권장합니다. 정수론 심화 문제나 경시대회 기출문제에 도전해보세요.
```

### Good (T-Score 60~69)

**코멘트:**
```
유리수 계산과 순환소수 변환 능력은 우수합니다. 기본 개념은 잘 이해하고 있으나, 복잡한 계산이나 여러 단계를 거치는 문제에서 일부 실수가 발생합니다. 소인수분해 활용 문제는 잘 해결하고 있습니다.
```

**학습 조언:**
```
계산 정확도 향상을 위한 반복 훈련이 필요합니다. 복잡한 분수식 계산 문제를 단계별로 꼼꼼히 풀이하는 연습을 하세요. 검산 습관을 기르면 실수를 줄일 수 있습니다.
```

### Average (T-Score 40~59)

**코멘트:**
```
기본적인 유리수 계산은 가능하나, 순환소수를 분수로 나타내거나 복잡한 분수식을 정리하는 과정에서 어려움을 겪고 있습니다. 소인수분해의 활용이 미흡합니다.
```

**학습 조언:**
```
순환소수와 분수의 관계를 다시 정리하고, 기본 계산 문제부터 차근차근 연습하세요. 약분과 통분 과정을 정확히 이해하고, 소인수분해를 활용한 최대공약수/최소공배수 문제를 반복 학습하세요.
```

### Weak (T-Score 30~39)

**코멘트:**
```
유리수의 기본 개념과 사칙연산에서 많은 오류가 발생하고 있습니다. 분수의 덧셈, 뺄셈, 곱셈, 나눗셈의 기본 규칙이 불완전하며, 순환소수 개념도 부족합니다.
```

**학습 조언:**
```
현재 학년 과정을 재학습해야 합니다. 분수의 사칙연산부터 차근차근 다시 시작하세요. 개념서를 활용하여 기본 원리를 이해하고, 쉬운 문제부터 충분히 연습한 후 단계를 높여가세요.
```

### Critical (T-Score 30 미만)

**코멘트:**
```
수와 연산의 기초 개념이 매우 부족합니다. 유리수의 정의부터 이해가 필요하며, 기본적인 분수 계산도 어려움을 겪고 있습니다. 현재 수준으로는 다음 단계 학습이 불가능합니다.
```

**학습 조언:**
```
이전 학년 내용부터 재학습이 시급합니다. 1:1 맞춤 지도나 소그룹 특별 관리가 필요합니다. 분수의 개념부터 천천히 다시 시작하여 기초를 탄탄히 다져야 합니다.
```

### 틀린 문항 패턴별 추가 코멘트

**복잡한 분수식 계산 오답 (3번, 6번 유형):**
```
복잡한 분수식에서 계산 실수가 발생했습니다. 단계를 나누어 차근차근 계산하고, 중간 과정을 반드시 적어가며 풀이하세요.
```

**소인수분해 활용 오답 (2번, 8번 유형):**
```
소인수분해를 활용한 문제에서 오답이 나왔습니다. 소인수분해의 개념과 최대공약수/최소공배수 구하는 방법을 다시 정리하세요.
```

## ➕ 영역 2: 식의 계산 (Algebraic Operations)

### Excellent (T-Score 70+)

**코멘트:**
```
단항식과 다항식의 사칙연산을 완벽하게 수행하며, 지수법칙 적용도 정확합니다. 복잡한 식의 전개와 정리 능력이 뛰어나 대수적 조작 능력이 매우 우수합니다. 고등 수학의 인수분해와 항등식으로 자연스럽게 확장할 수 있는 탄탄한 기초를 갖추고 있습니다.
```

**학습 조언:**
```
현재 수준을 유지하며 고등 과정의 다항식 전개와 인수분해로 확장하세요. 특히 곱셈 공식의 변형과 복잡한 인수분해 문제에 도전해보세요.
```

### Good (T-Score 60~69)

**코멘트:**
```
기본적인 식의 계산은 정확하게 수행합니다. 단항식의 곱셈과 나눗셈, 다항식의 덧셈과 뺄셈은 잘 해결하고 있으나, 복잡한 전개식이나 여러 단계를 거치는 식 정리에서 일부 실수가 있습니다.
```

**학습 조언:**
```
복잡한 식을 단계별로 정리하는 연습이 필요합니다. 괄호가 여러 개 있는 식의 전개와 동류항 정리를 반복 학습하세요. 부호 실수를 줄이기 위해 중간 과정을 꼼꼼히 확인하세요.
```

### Average (T-Score 40~59)

**코멘트:**
```
기본 연산은 이해하고 있으나, 지수법칙 적용이나 복잡한 식의 정리에서 어려움을 겪고 있습니다. 특히 음수와 분수가 포함된 식의 계산에서 오류가 많이 발생합니다.
```

**학습 조언:**
```
지수법칙의 기본 규칙을 다시 정리하고, 단순한 문제부터 반복 연습하세요. 부호 규칙과 분배법칙을 확실히 이해하고 적용하는 연습이 필요합니다.
```

### Weak (T-Score 30~39)

**코멘트:**
```
식의 계산 기본 규칙이 불완전합니다. 단항식의 곱셈과 나눗셈에서도 오류가 발생하며, 지수법칙의 이해가 부족합니다. 다항식의 전개와 정리가 거의 불가능한 상태입니다.
```

**학습 조언:**
```
식의 계산 기초부터 재학습이 필요합니다. 지수의 의미부터 다시 이해하고, 간단한 단항식 계산부터 시작하세요. 분배법칙을 완전히 숙지할 때까지 반복 연습하세요.
```

### Critical (T-Score 30 미만)

**코멘트:**
```
문자와 식의 기본 개념이 매우 부족합니다. 단항식과 다항식의 구분도 어려워하며, 기본적인 식의 계산조차 수행하기 어려운 상태입니다.
```

**학습 조언:**
```
현재 과정을 재이수해야 합니다. 문자의 의미와 대입의 개념부터 천천히 학습하세요. 1:1 개별 지도를 통해 기초부터 차근차근 다져야 합니다.
```

### 틀린 문항 패턴별 추가 코멘트

**지수법칙 오답 (11번, 6번 유형):**
```
지수법칙 적용에서 오답이 나왔습니다. 곱셈과 나눗셈에서 지수를 어떻게 처리하는지 규칙을 다시 정리하세요.
```

**복잡한 전개식 오답 (14번 유형):**
```
복잡한 전개식 계산에서 실수했습니다. 분배법칙을 단계별로 적용하고, 중간 과정을 생략하지 마세요.
```

## 📐 영역 3: 방정식 (Equations)

### Excellent (T-Score 70+)

**코멘트:**
```
일차방정식과 연립방정식을 완벽하게 해결하며, 복잡한 문장제 문제도 정확히 식으로 전환하여 풀이합니다. 방정식의 해를 구하는 과정이 정확하고, 검산을 통한 확인도 철저합니다. 고등 과정의 이차방정식과 고차방정식으로 자연스럽게 확장할 수 있습니다.
```

**학습 조언:**
```
현재 수준을 유지하며 고등 과정의 이차방정식, 연립이차방정식으로 확장하세요. 절댓값이 포함된 방정식이나 무리방정식 등 심화 문제에 도전해보세요.
```

### Good (T-Score 60~69)

**코멘트:**
```
기본적인 방정식 풀이 능력은 우수합니다. 일차방정식과 연립방정식의 해법은 잘 이해하고 있으나, 복잡한 문장제 문제를 식으로 세우거나 특수한 형태의 방정식에서 일부 어려움이 있습니다.
```

**학습 조언:**
```
문장제 문제를 식으로 전환하는 연습이 필요합니다. 문제에서 미지수가 무엇을 의미하는지 명확히 하고, 조건을 식으로 나타내는 훈련을 하세요. 특수한 형태의 방정식 유형도 학습하세요.
```

### Average (T-Score 40~59)

**코멘트:**
```
기본 방정식은 풀 수 있으나, 연립방정식의 소거법이나 대입법 적용이 미흡합니다. 문장제 문제를 식으로 세우는 데 어려움을 겪고 있으며, 복잡한 계수가 있는 방정식에서 계산 실수가 많습니다.
```

**학습 조언:**
```
연립방정식의 소거법과 대입법을 다시 정리하고 반복 연습하세요. 문장제 문제는 문제를 여러 번 읽고, 주어진 조건을 하나씩 식으로 나타내는 연습이 필요합니다.
```

### Weak (T-Score 30~39)

**코멘트:**
```
방정식의 기본 풀이 과정에서 많은 오류가 발생합니다. 이항과 양변을 같은 수로 나누는 과정에서 실수가 많으며, 연립방정식 풀이가 거의 불가능한 상태입니다.
```

**학습 조언:**
```
방정식의 기본 개념부터 재학습하세요. 등식의 성질을 완전히 이해하고, 간단한 일차방정식부터 충분히 연습한 후 연립방정식으로 넘어가세요.
```

### Critical (T-Score 30 미만)

**코멘트:**
```
방정식의 기초 개념이 매우 부족합니다. 등식의 의미와 미지수의 개념 이해가 필요하며, 기본적인 방정식 풀이조차 어려워하고 있습니다.
```

**학습 조언:**
```
이전 학년 내용부터 재학습이 필요합니다. 등식과 부등식의 차이, 미지수의 의미부터 천천히 학습하세요. 1:1 맞춤 지도가 필요합니다.
```

### 틀린 문항 패턴별 추가 코멘트

**연립방정식 소거법 오답 (15번 유형):**
```
연립방정식 소거법에서 오답이 나왔습니다. 두 식을 더하거나 빼서 한 미지수를 소거하는 과정을 다시 연습하세요.
```

**복잡한 계수 방정식 오답 (16번 유형):**
```
계수가 복잡한 방정식에서 실수했습니다. 분모를 없애기 위해 양변에 최소공배수를 곱하는 과정을 정확히 수행하세요.
```

## 📈 영역 4: 함수 (Functions)

### Excellent (T-Score 70+)

**코멘트:**
```
일차함수의 개념을 완벽히 이해하고 있으며, 그래프 해석과 활용 능력이 매우 뛰어납니다. 함수의 식을 구하고, 그래프를 그리며, 실생활 문제에 적용하는 능력이 우수합니다. 기울기와 절편의 의미를 정확히 파악하고 있습니다.
```

**학습 조언:**
```
현재 수준을 유지하며 고등 과정의 이차함수로 확장하세요. 함수의 평행이동, 대칭이동 등 변환에 대한 심화 학습과 최대·최소 문제에 도전해보세요.
```

### Good (T-Score 60~69)

**코멘트:**
```
일차함수의 기본 개념은 잘 이해하고 있으며, 간단한 그래프 해석과 식 구하기는 정확합니다. 그러나 복잡한 응용 문제나 여러 함수가 결합된 문제에서 일부 어려움이 있습니다.
```

**학습 조언:**
```
그래프 해석 문제를 다양하게 풀어보세요. 특히 두 일차함수의 교점, 넓이, 도형과 결합된 문제 유형을 연습하세요. 기울기의 의미를 더 깊이 이해하는 것이 중요합니다.
```

### Average (T-Score 40~59)

**코멘트:**
```
일차함수의 기본 정의는 이해하고 있으나, 그래프를 그리거나 해석하는 능력이 부족합니다. 기울기와 y절편의 의미를 혼동하며, 함수식을 구하는 문제에서 어려움을 겪고 있습니다.
```

**학습 조언:**
```
일차함수의 기본 개념을 다시 정리하세요. y = ax + b 형태에서 a와 b가 그래프에서 어떤 의미인지 확실히 이해하고, 간단한 그래프 그리기부터 반복 연습하세요.
```

### Weak (T-Score 30~39)

**코멘트:**
```
일차함수의 기본 개념이 불완전합니다. 함수의 정의부터 이해가 부족하며, 그래프와 식의 관계를 파악하지 못하고 있습니다. 기울기와 절편의 개념이 매우 약합니다.
```

**학습 조언:**
```
함수의 개념부터 재학습이 필요합니다. 좌표평면에 점을 찍는 것부터 시작하여, 정비례와 반비례 관계를 다시 정리한 후 일차함수로 넘어가세요.
```

### Critical (T-Score 30 미만)

## 🎯 득점 향상 예측 공식

### 기본 공식

```
예상 득점 = (영역 배점) × (목표 달성률) × (학습 기간 계수)

학습 기간 계수:
- 1~2개월: 0.5~0.6
- 2~3개월: 0.7~0.8
- 3개월+: 0.9~1.0

목표 달성률:
- Critical → Weak: 30~40%
- Weak → Average: 50~60%
- Average → Good: 70~80%
- Good → Excellent: 85~95%
```

### 예시 계산

**학생 상황:** 함수 영역 T-Score 32 (Critical), 배점 36.5점, 현재 정답률 44%

**1단계 목표:** Critical → Average (정답률 50~60% 달성)

**예상 득점 향상:**
```
현재 득점: 36.5 × 0.44 = 16.1점
목표 득점: 36.5 × 0.55 = 20.1점
득점 향상: 20.1 - 16.1 = 4점 (보수적 예측)

또는

현재 득점: 36.5 × 0.44 = 16.1점  
목표 득점: 36.5 × 0.60 = 21.9점
득점 향상: 21.9 - 16.1 = 5.8점 (낙관적 예측)

→ 표시: +4~6점
```

## 🔧 구현 시 고려사항

1. **우선순위 로직**
   ```python
   # 취약 영역 우선순위 결정
   def get_priority_areas(results):
       areas = []
       for area, data in results.items():
           if data['t_score'] < 40:  # Critical
               priority = 1
           elif data['t_score'] < 60:  # Weak
               priority = 2
           else:
               priority = 3
           
           areas.append({
               'name': area,
               'priority': priority,
               'weight': data['total_score']  # 배점
           })
       
       # 우선순위 -> 배점 순으로 정렬
       return sorted(areas, key=lambda x: (x['priority'], -x['weight']))
   ```

2. **틀린 문항 패턴 분석**
   ```python
   def analyze_wrong_patterns(area, wrong_items):
       patterns = []
       
       # 고난도 집중 오답
       if set(wrong_items) >= set([21, 22, 23, 24, 25]):
           patterns.append("고난도_전체오답")
       
       # 특정 유형 오답
       if area == "함수" and 17 in wrong_items and 18 in wrong_items:
           patterns.append("그래프_해석_오답")
       
       return patterns
   ```

3. **득점 향상 예측**
   ```python
   def predict_score_gain(area, current_score, target_level, period):
       area_weight = AREA_WEIGHTS[area]  # 배점
       current_rate = current_score / area_weight
       
       # 목표 달성률
       target_rates = {
           'critical_to_weak': 0.35,
           'weak_to_average': 0.55,
           'average_to_good': 0.75
       }
       
       target_rate = target_rates[target_level]
       period_coef = PERIOD_COEF[period]  # 1~2개월: 0.55
       
       estimated_gain = area_weight * (target_rate - current_rate) * period_coef
       
       # 보수적/낙관적 범위
       conservative = estimated_gain * 0.8
       optimistic = estimated_gain * 1.2
       
       return f"+{int(conservative)}~{int(optimistic)}점"
   ```
