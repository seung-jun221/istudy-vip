## 💡 등급 해석 가이드

### 학생/학부모용 안내

#### ✅ 1-2등급 (A등급)
**현재 수준**: 최상위권
**고등 예상**: 1-2등급
**학습 방향**:
- 심화 문제 집중 훈련
- 고등 선행 진행 권장
- 경시/올림피아드 도전

#### ✅ 3-4등급 (B등급)
**현재 수준**: 중상위권
**고등 예상**: 2-4등급
**학습 방향**:
- 기본 개념 완벽 정리
- 중난도 문제 반복 훈련
- 약점 영역 집중 보완

#### ⚠️ 5-6등급 (C등급)
**현재 수준**: 평균 이하
**고등 예상**: 4-6등급
**학습 방향**:
- 기초 개념부터 재정립
- 기본 문제 완벽 숙지
- 학습 습관 개선 필수

#### 🚨 7-9등급 (D-E등급)
**현재 수준**: 기초 부족
**고등 예상**: 6-9등급
**학습 방향**:
- 현재 과정 재학습
- 1:1 맞춤 지도 필요
- 학습 동기 부여 우선

   - 등급별 분포

3. **조정 방법**:
   - 실제 평균이 예상과 5점 이상 차이 시 등급컷 재조정
   - 표준편차가 예상보다 5점 이상 차이 시 등급 구간 재설정
   - 매년 고1 9월 모의고사 결과와 비교 검증
